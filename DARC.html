Project Overview

Erik Contreras has been developing an autonomous racing car in Assistant Professor of Mechanical and Aerospace Engineering Shima Nazari's lab for preparation of F1tenth competition. Each year, Erik worked with UC Davis students to implement new features to the car. In 2024, the car integrated localization system using LiDAR system and succeeded in running autonomously when the predefined race map was given. In 2025, my team and I from ECS 193 (capstone project) started working with Erik to integrate computer vision to the system.

Project Demo

Project Goals

  • Develop a fully autonomous racing vehicle
  • Implement real-time object detection and tracking
  • Create efficient path planning algorithms

Key Features

  • Recognition of racetrack in Bird-Eye View (BEV) using post data processing with segmentation model and perspective transform.
  • Real time data processing using neural network system.
  • Autonomous navigation and path planning.

Technical Details

  • Segmentation Models: Lang Segment Anything Model (LangSAM), Tiny Segment Anything Model (TinySAM)
  • Perspective Transform: OpenCV
  • Custom-built image fragment stitching
  • Algorithms for emergency breaks and obstacle avoidance
